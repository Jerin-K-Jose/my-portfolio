Project Overview

The Human Activity Recognition (HAR) using Radar project leverages radar technology and machine learning to classify human activities. This innovative approach eliminates the need for wearable devices, making it more convenient and less intrusive. The system can also be used for applications such as health monitoring, security systems, and smart environments.

We used a publicaly available dataset named uRadHAR dataset containing micro-Doppler signatures of different human activities performed by different people. These data are then processed using signal processing techniques to extract useful features, which were used to train and classify using a BiLSTM model implemented in MATLAB.
